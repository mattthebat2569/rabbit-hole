Proof that sqrt(2) is Irrational using Induction

by Zero Two, Apr 13, 2021, 9:39 AM

We wish to prove for all positive integers $n$ that there is no positive integer $a$ such that $\sqrt2 = \frac{n}{a}$. Let $f(n)$ be the statement such that there is no positive integer $a$ such that $\sqrt2 = \frac{n}{a}$. Base Step: $f(1)$ is true because $\sqrt2 > 1 \geq \frac{1}{a}$ for all positive integers $a$. Inductive Step: Assume that $f(x)$ is true for all $x \leq k$, where $k$ is some positive integer. We will now prove that $f(k+1)$ is true by contradiction. Assume that $\sqrt2 = \frac{k+1}{a}$. We then have \[a\sqrt2 = k + 1\]\[2a^2 = (k+1)^2\]. This implies $(k+1)^2$ is even and so $(k+1)$ is also even. We can let $(k+1)$ be $2i$, where $i$ is a positive integer. Then we have \[2a^2 = (2i)^2\]\[2a^2 = 4i^2\]\[a^2 = 2i^2 \]. This tells us that $a$ is even and so we can let $a = 2t$, for some positive integer $t$. We then substitute our values for $a$ and $k+1$ back into the original equation. \[\sqrt2 = \frac{2i}{2s}\]\[\sqrt2 = \frac{i}{s}\]. But since $i \leq k$, we have a contradiction and so we are done. $\blacksquare$
